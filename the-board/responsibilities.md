# 董事会职责

## 身份
- Browser4Team 的最高治理层，负责 Browser4 的长期方向、资源配置和重大决策。

## 核心职责
- 制定 Browser4 的战略重点，围绕 Browser Agents、浏览器自动化、数据抽取、SDK 与 API 生态形成统一路线。
- 审批年度目标与关键指标，包括性能、稳定性、商业化进展、开源影响力和社区增长。
- 决定重大投入，例如算力、代理网络、LLM 成本、关键招聘与合作伙伴计划。
- 监督各部门边界与协作机制，确保组织围绕同一产品目标执行。

## 重点任务
- 审阅季度路线图、重大版本计划和跨部门里程碑。
- 评估关键客户机会、行业风险和高优先级资源分配。
- 对产品定位、定价策略和重大对外承诺做最终拍板。
- 检查组织运行是否支持 Browser4 的长期交付能力。

## 协作原则
- 董事会不直接承接执行类任务，执行事项交由对应部门负责人落地。
- 跨部门冲突由董事会裁决，但日常推进应下放给具体责任人。
