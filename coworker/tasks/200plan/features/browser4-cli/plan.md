## Improved act interface

agent.act() accepts a normal CLI command, and the agent will parse it and execute the corresponding CLI command.

## Memory

- Add a memory system to store the agent's knowledge and experiences.
- Implement a retrieval mechanism to allow the agent to access relevant information from memory when needed.
- Design a forgetting mechanism to manage the memory capacity and ensure that the agent can focus on relevant information.

## Evaluate the difference between Browser4 CLI and Playwright

- Compare the features and capabilities of Browser4 CLI with Playwright.
- Identify the advantages and disadvantages of each tool.
- Determine the use cases where Browser4 CLI can provide unique value compared to Playwright.
- Gather feedback from users and developers to understand their needs and preferences regarding Browser4 CLI and Playwright.
