# CLI: browser4-cli install

Add a `browser4-cli install` command that installs the browser environment. The command should verify that Java, Chrome, and `Browser4.jar` are available, and install any missing components automatically.

Users can download `browser4-cli.exe` or `browser4-cli` from the releases page and run:

```shell
browser4-cli install
```
