# Safe File Moving Script

Write a script to move ps1/sh scripts from a source directory to a destination directory. The script should ensure that it does not overwrite any existing files in the destination directory.
If a file with the same name already exists, the script should append a timestamp to the filename before moving it.

Make sure all the references to the scripts in the source directory are updated to point to the new location in the destination directory.
