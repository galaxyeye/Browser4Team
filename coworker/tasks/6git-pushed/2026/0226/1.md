# 优化 coworker 输出目录

Coworker.ps1/sh 的日志输出到 coworker/tasks/300logs 目录下。

Write output in 300logs with date-time-based organization, a directory each day, and files named with time, e.g. 012530file1.log, 160325file2.log.

date-time-based organization:

- 300logs/
    - 2026/
        - 0225/
            - 012530file1.log
            - 160325file2.log
