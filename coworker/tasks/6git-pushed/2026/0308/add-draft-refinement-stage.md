# Improve Coworker Workflow

Add a new stage to refine the draft automatically generated by the AI.

This stage will involve a review process where the draft is evaluated for clarity, coherence, and relevance to the intended audience.
The reviewer will provide feedback and suggestions for improvement, which will then be incorporated into the final version of the document.
This will help ensure that the content is of high quality and meets the needs of the target audience.

## Workflow

- Create a script to refine the given file or all files in a folder
    - follow refine-last-draft.ps1 pattern to refine the draft
    - default scanning folder is `tasks/0draft/refine/1ready`
    - scan all files in the folder, and refine them one by one, move them to `2working` dir to avoid duplicate processing, and move the refined version to `3done` dir
- Create a script to run the refine script periodically or on demand
