# Analyze Directory Metadata - Optimized Version

Improve analyze-directory-metadata.ps1, do not analyze subdirectories by default to enhance performance. 
Instead, provide an option to include subdirectories when needed. This way, users can quickly analyze the main directory 
without waiting for a full scan of all subdirectories, which can be time-consuming for large directories.
