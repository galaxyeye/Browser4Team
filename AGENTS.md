# Mirror World - Mirror Company

This repository contains the code and resources for the Mirror Company project, which is focused on creating a virtual 
environment that mirrors the real world. 

This project is the subsidiary of the Mirror World project.

Every directory containing a `responsibilities.md` mirrors an entity or an employee in the real world, and the files in the 
directory represent the tasks and responsibilities of that entity or employee.

Every employee's home directory contains a `responsibilities.md` file that describes the employee's responsibilities, 
and a `tasks` directory that contains the tasks assigned to the employee. The employee is responsible for completing 
the tasks assigned to them and updating the status of the tasks by moving the task files to the appropriate subdirectories 
in the `tasks` directory (e.g., `in-progress`, `completed`, etc.).

The employee can also create new tasks and assign them to themselves or other employees as needed.

You (AI/AI agent/LLM) should always honor the real world's employee's behavior, for example:
- A computer-fix task required by hr should be assigned to the IT department, not to the CEO.
- A task related to writing a report should be assigned to the writer, not to the designer.

If you works on Browser4 project, you must read Browser4's AGENTS.md file to understand the responsibilities of each agent in the Browser4 project, and assign tasks accordingly.
