# EventBus Observability Mechanism

## Overview

The EventBus mechanism has been added to provide better observability and testability for the PerceptiveAgent and InferenceEngine components. Events are emitted at key points before and after major operations, allowing external listeners to monitor, log, and test the system's behavior.

## Event Types

### PerceptiveAgent Events

The following events are emitted by PerceptiveAgent methods:

- **run**
  - `PerceptiveAgent.onWillRun` - Before execution starts
  - `PerceptiveAgent.onDidRun` - After execution completes

- **observe**
  - `PerceptiveAgent.onWillObserve` - Before observation starts
  - `PerceptiveAgent.onDidObserve` - After observation completes

- **act**
  - `PerceptiveAgent.onWillAct` - Before action execution
  - `PerceptiveAgent.onDidAct` - After action execution

- **extract**
  - `PerceptiveAgent.onWillExtract` - Before extraction starts
  - `PerceptiveAgent.onDidExtract` - After extraction completes

- **summarize**
  - `PerceptiveAgent.onWillSummarize` - Before summarization starts
  - `PerceptiveAgent.onDidSummarize` - After summarization completes

### InferenceEngine Events

The following events are emitted by InferenceEngine methods:

- **observe**
  - `InferenceEngine.observe.willGenerate` - Before LLM generation
  - `InferenceEngine.observe.didGenerate` - After LLM generation

- **extract**
  - `InferenceEngine.onWillExtract` - Before extraction starts
  - `InferenceEngine.onDidExtract` - After extraction completes

- **summarize**
  - `InferenceEngine.onWillSummarize` - Before summarization starts
  - `InferenceEngine.onDidSummarize` - After summarization completes

## Usage Examples

### Example 1: Monitoring Action Execution

```kotlin
import ai.platon.pulsar.skeleton.workflow.EventBus

// Register a handler to monitor action execution
DangerousEventBus.register("PerceptiveAgent.onWillAct") { payload ->
    val map = payload as? Map<String, Any?> ?: return@register null
    val action = map["action"]
    println("Starting action: $action")
    payload
}

DangerousEventBus.register("PerceptiveAgent.onDidAct") { payload ->
    val map = payload as? Map<String, Any?> ?: return@register null
    val result = map["result"]
    println("Action completed: $result")
    payload
}

// Clean up when done
EventBus.unregister("PerceptiveAgent.onWillAct")
EventBus.unregister("PerceptiveAgent.onDidAct")
```

### Example 2: Collecting Metrics

```kotlin
import ai.platon.pulsar.skeleton.workflow.EventBus
import java.time.Instant

val metrics = mutableMapOf<String, MutableList<Long>>()

// Track execution times
DangerousEventBus.register("PerceptiveAgent.onWillObserve") { payload ->
    val map = payload as? Map<String, Any?> ?: return@register null
    val uuid = map["uuid"] as? String
    if (uuid != null) {
        metrics.computeIfAbsent("observe-$uuid") { mutableListOf() }
            .add(Instant.now().toEpochMilli())
    }
    payload
}

DangerousEventBus.register("PerceptiveAgent.onDidObserve") { payload ->
    val map = payload as? Map<String, Any?> ?: return@register null
    val uuid = map["uuid"] as? String
    if (uuid != null) {
        val times = metrics["observe-$uuid"]
        if (times != null && times.isNotEmpty()) {
            val duration = Instant.now().toEpochMilli() - times.last()
            println("Observation took ${duration}ms")
        }
    }
    payload
}
```

### Example 3: Validating ActionDescription

```kotlin
import ai.platon.pulsar.skeleton.workflow.EventBus
import ai.platon.pulsar.agentic.model.ActionDescription

// Validate action descriptions generated by LLM
DangerousEventBus.register("InferenceEngine.observe.didGenerate") { payload ->
    val map = payload as? Map<String, Any?> ?: return@register null
    val actionDescription = map["actionDescription"] as? ActionDescription

    if (actionDescription != null) {
        // Perform validation
        require(actionDescription.method != null) {
            "ActionDescription must have a method"
        }
        println("Generated action: ${actionDescription.method}")

        // You can also modify or enrich the payload
        val enriched = map.toMutableMap()
        enriched["validated"] = true
        return@register enriched
    }

    payload
}
```

### Example 4: Testing with EventBus

```kotlin
import ai.platon.pulsar.skeleton.workflow.EventBus
import org.junit.jupiter.api.Test
import org.junit.jupiter.api.Assertions.*

class MyAgentTest {

    @Test
    fun testAgentEmitsEvents() {
        val capturedEvents = mutableListOf<Map<String, Any?>>()

        // Register handler to capture events
        DangerousEventBus.register("PerceptiveAgent.onWillRun") { payload ->
            capturedEvents.add(payload as Map<String, Any?>)
            payload
        }

        // Simulate emitting an event
        val testPayload = mapOf(
            "action" to "test action",
            "uuid" to "test-uuid"
        )
        DangerousEventBus.emit("PerceptiveAgent.onWillRun", testPayload)

        // Give EventBus time to process
        Thread.sleep(100)

        // Verify event was captured
        assertTrue(capturedEvents.isNotEmpty())
        assertEquals("test action", capturedEvents[0]["action"])

        // Clean up
        EventBus.unregister("PerceptiveAgent.onWillRun")
    }
}
```

### Example 5: Logging All Events

```kotlin
import ai.platon.pulsar.skeleton.workflow.EventBus
import org.slf4j.LoggerFactory

val logger = LoggerFactory.getLogger("EventBusLogger")

// Register a generic logger for all agent events
listOf(
    "PerceptiveAgent.onWillRun",
    "PerceptiveAgent.onDidRun",
    "PerceptiveAgent.onWillObserve",
    "PerceptiveAgent.onDidObserve",
    "PerceptiveAgent.onWillAct",
    "PerceptiveAgent.onDidAct",
    "PerceptiveAgent.onWillExtract",
    "PerceptiveAgent.onDidExtract",
    "PerceptiveAgent.onWillSummarize",
    "PerceptiveAgent.onDidSummarize"
).forEach { eventType ->
    DangerousEventBus.register(eventType) { payload ->
        logger.info("Event: $eventType, Payload: $payload")
        payload
    }
}
```

## Payload Structure

Each event type has a specific payload structure:

### PerceptiveAgent Events

- **run**: `{ action: ActionOptions, uuid: UUID, result?: AgentHistory, stateHistory?: AgentHistory }`
- **observe**: `{ options: ObserveOptions, uuid: UUID, observeResults?: List<ObserveResult>, actionDescription?: ActionDescription }`
- **act**: `{ action: ActionOptions, uuid: UUID, result?: ActResult }`
- **extract**: `{ options: ExtractOptions, uuid: UUID, result?: ExtractResult }`
- **summarize**: `{ instruction: String?, selector: String?, uuid: UUID, result?: String }`

### InferenceEngine Events

- **observe**: `{ context: ExecutionContext, messages: AgentMessageList, actionDescription?: ActionDescription }`
- **extract**: `{ params: ExtractParams, result?: ObjectNode, extractedNode?: ObjectNode, metaNode?: ObjectNode }`
- **summarize**: `{ instruction: String?, textContentLength: Int, result?: String, tokenUsage?: TokenUsage }`

## Best Practices

1. **Always check payload type**: Cast the payload safely with `as?` to avoid ClassCastException
2. **Return the payload**: Handlers should return the payload (or a modified version) to allow chaining
3. **Keep handlers lightweight**: EventBus runs asynchronously, but heavy processing can impact performance
4. **Clean up handlers**: Unregister handlers when they're no longer needed to prevent memory leaks
5. **Use for observability**: EventBus is great for monitoring, logging, and testing, not for business logic

## Implementation Details

- Events are emitted asynchronously using Kotlin coroutines
- Only the last registered handler for each event type is active (handlers overwrite)
- Event handlers are executed in a background coroutine scope
- Handlers should be thread-safe as they may be called concurrently
