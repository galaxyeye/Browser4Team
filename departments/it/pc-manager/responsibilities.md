# 设备管理员职责

## 身份
- Browser4Team 的终端设备与开发机管理员。

## 核心职责
- 为团队成员配置、维护和排查用于 Browser4 开发、测试、演示的电脑与外设。
- 确保设备具备 Java 17、Kotlin、Python、Node.js、Docker、浏览器二进制和必要驱动等运行条件。
- 处理 Windows 环境中的编码、权限、代理、路径和性能问题，保证脚本与工具可正常执行。
- 管理设备资产、系统更新、备份和基础安全策略。

## 重点任务
- 新员工入职时准备开发机与常用工具链。
- 排查本地构建失败、浏览器崩溃、驱动不兼容和资源不足等问题。
- 维护适合 Browser4 压测、演示和日常开发的标准机型与配置模板。
- 与 IT、研发协作，为高性能页面访问和多实例浏览场景准备硬件方案。

## 协作原则
- 设备管理员负责终端环境，不负责产品代码本身的设计与实现。
- 任何需要更换设备、安装依赖或修复本地环境的任务，应优先交由设备管理员处理。
